# Air724UG 综合工具

这是一个整合了Air724UG短信助手和串口监控工具的综合应用程序。

## 功能特点

1. **双端口管理**：同时连接短信助手端口和监控工具端口
2. **设备状态显示**：当两个端口都连接成功时，显示设备连接成功，并将读取到的手机号码作为设备名称
3. **整合日志系统**：支持显示全部日志、仅短信助手日志或仅监控工具日志
4. **短信发送功能**：可发送短信到指定手机号码
5. **SIM卡信息读取**：读取并显示SIM卡的手机号码和运营商信息
6. **自动端口检测**：自动检测并选择合适的串口端口
7. **自动复制验证码**：可自动提取短信中的验证码并复制到剪贴板
8. **智能乱码修复**：自动处理并修复短信内容中的乱码问题
9. **功能状态提醒**：勾选自动复制验证码等功能时，日志框会显示相应提醒

## 安装说明

1. 确保已安装Python 3.6或更高版本
2. 安装必要的依赖包：
   ```
   pip install -r requirements.txt
   ```
3. 直接运行 `run_combined_tool.bat` 启动程序

## 使用方法

1. **端口连接**：
   - 程序启动后会自动检测并选择合适的端口
   - 点击"连接"按钮连接短信助手端口
   - 点击"打开串口"按钮连接监控工具端口
   - 当两个端口都连接成功时，设备名称会显示为读取到的手机号码

2. **日志查看**：
   - 通过选择日志类型，可以查看全部日志、仅短信助手日志或仅监控工具日志
   - 点击"清除日志"按钮可以清空当前显示的日志

3. **短信功能**：
   - 输入目标手机号码和短信内容
   - 点击"发送短信"按钮发送短信
   - 程序会显示发送统计信息

4. **SIM卡信息**：
   - 点击"读取SIM卡信息"按钮获取SIM卡的手机号码和运营商信息

5. **自动复制验证码**：
   - 勾选"自动复制验证码"复选框开启自动复制功能
   - 当收到包含验证码的短信时，系统会自动提取验证码并复制到剪贴板
   - 日志框会显示"已启用自动复制验证码功能"的提醒

6. **短信收件箱**：
   - 系统会自动收集并显示接收到的短信
   - 点击"刷新收件箱"按钮可以重新读取所有短信
   - 系统会自动处理短信内容中的乱码问题，确保显示清晰可读的短信内容

## 注意事项

1. 使用前请确保Air724UG模块正确连接到电脑
2. 确保安装了正确的驱动程序
3. 短信中心号码可能需要根据地区进行修改
4. 发送短信前请确保SIM卡已激活并有足够的余额
5. 自动复制验证码功能目前支持常见的验证码格式，对于特殊格式的验证码可能无法正确识别
6. 乱码修复功能会尽可能还原短信内容，但对于严重损坏的短信可能无法完全修复
7. 为了获得更好的使用体验，建议定期清理日志文件

## 项目文件说明

以下是项目文件夹中各个文件的作用说明：

| 文件名 | 作用描述 |
|--------|---------|
| **README.md** | 项目的说明文档，包含功能介绍、安装指南和使用方法等信息。 |
| **combined_air724ug_tool.code-workspace** | Visual Studio Code的工作区配置文件，用于保存项目的编辑器设置、调试配置等。 |
| **combined_gui.py** | 主程序文件，包含了所有的功能实现，如双端口管理、短信发送、设备状态显示、自动复制验证码等核心功能。 |
| **combined_gui.spec** | PyInstaller打包配置文件，用于将Python代码打包成独立的可执行文件(.exe)。配置中指定了入口文件、依赖项、打包选项等。 |
| **requirements.txt** | 项目依赖文件，列出了运行程序所需的Python包及其版本。目前包含了pyserial包，用于串口通信。 |
| **run_combined_tool.bat** | Windows批处理文件，提供了一个便捷的方式启动程序。双击该文件会自动在当前目录下运行Python程序。 |
| **venv\** | Python虚拟环境文件夹，包含了项目独立的Python解释器和安装的依赖包，确保项目运行环境的一致性。
